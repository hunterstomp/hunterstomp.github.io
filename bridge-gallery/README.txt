Bridge Gallery – UX Portfolio Tool

This package includes:
- /images/thumbs/          → 300px thumbnails
- /images/optimized/       → Optimized full-size images
- gallery.json             → Curated metadata
- gallery_autogenerated.json → Auto-generated from filenames
- bridge_gallery.html      → HTML version with smart filtering
- BridgeGallery.jsx        → React component
- README.txt               → This file

To use:
1. Open bridge_gallery.html in a browser for static use
2. Use BridgeGallery.jsx in any React app (Tailwind CSS included)
3. Customize gallery.json to update tags/categories
4. Optional: Deploy with Netlify, GitHub Pages, or Vercel

Enjoy!
