T-Mobile Prospect Chat Experience

Project Summary

Client: T-Mobile (ABC)

Role: UX Designer, Journey Mapper

Team: Product, Engineering, Care Ops, Retail Experts

Platform: Web + iOS

Problem

T-Mobile needed a seamless online-to-retail experience that would reduce churn, personalize support across channels, and increase mobile chat conversions.

Goal

Create an end-to-end journey that feels human, supports configuration within chat, and eliminates trust/timing blockers.

Research & Discovery

  • User flows from chat and bot handoffs
  • Google Shopping and Microsite UX analysis
  • Journey maps (ABC, IoY, HOT SIM)
“This feels very personalized :)”

UX Approach

  • Designed live chat + bot handoff interface with indicators
  • Embedded Terms CTA with re-entry guidance
  • Finance picker & Apple Pay chat integration
