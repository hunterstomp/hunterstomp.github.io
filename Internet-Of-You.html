Best Practices for UX Case Study Image Optimization

  • Format Selection: Use PNG for UIs, SVG for diagrams, and JPG for photos.
  • File Size: Compress images with tools like TinyPNG or Squoosh.
  • Dimensions: Standardize gallery images to 4:3 or 16:9 aspect ratios.
  • Annotations: Add arrows or blur backgrounds to focus attention.
  • Lazy Loading: Use loading="lazy" for performance.
  • Alt Text: Write purpose-driven descriptions for accessibility.

Suggested Dimensions

  • Slideshows: 1920x1080
  • Mobile screens: 750x1334
  • Wireframes: 1280x800 or scalable SVG

Accessibility Note

Each image should use descriptive alt attributes that focus on the user experience impact rather than generic labels. E.g.:

alt="Chat handoff from bot to live agent during T-Mobile checkout"
